# Mini Java Projects Repository

## Overview

This repository contains a collection of **Mini Java Projects** designed to help students and beginners learn core Java programming concepts through practical implementation. The projects cover topics such as object-oriented programming, file handling, collections, networking, GUI development, and database connectivity.

## Objectives

* Strengthen Java programming skills.
* Understand real-world application development.
* Practice problem-solving and coding techniques.
* Learn software design and project organization.

## Prerequisites

Before running the projects, ensure you have:

* Java Development Kit (JDK) 8 or later
* An IDE such as Eclipse, IntelliJ IDEA, or NetBeans
* Basic knowledge of Java programming
* Git (optional, for cloning the repository)

## Sample Projects

### 1. Calculator

A simple calculator that performs basic arithmetic operations.

**Features**

* Addition
* Subtraction
* Multiplication
* Division

### 2. Student Management System

Manages student records efficiently.

**Features**

* Add students
* Update records
* Delete students
* Search students

### 3. Library Management System

Helps manage books and library members.

**Features**

* Book issue and return
* Member management
* Search books

### 4. Chat Application

A socket-based client-server chat application.

**Features**

* Real-time messaging
* TCP socket communication
* Client-server architecture

### 5. Banking System

Simulates basic banking operations.

**Features**

* Deposit money
* Withdraw money
* Balance inquiry

### 6. Tic Tac Toe Game

A console-based two-player game.

**Features**

* Interactive gameplay
* Winner detection
* Draw condition handling

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/mini-java-projects.git
```

2. Navigate to the project folder:

```bash
cd mini-java-projects
```

3. Open the desired project in your IDE.

4. Compile and run the main Java file:

```bash
javac Main.java
java Main
```
